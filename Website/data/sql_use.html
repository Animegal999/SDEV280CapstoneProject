Results using SQL via PHPMyAdmin

SQL stands for Structured Query Language, and is primarily used in databases. For the purposes of this project, I used an application that can filter through databases that you can manually create called Microsoft Access.

For Access, we'll need to make our database. The tabular design is similar to Excel, so copy and pasting the columns into Access works just fine. This is what our table looks like, inside the database:



First off, there is an extra column that wasn't present in the Excel data. This column is necessary in MS Access, as it is an ID for each survey applicant. Something called a Primary Key is stored within this column. Primary keys make sure that:

  • Our data doesn't have null values
  • Our data doesn't repeat
  • Our data shows relationships within data tables that we can create
  • Our data can be easily updated with the inclusion of a primary key.

With our database, something that is cool that Access can do is use SQL to filter and sort through data. First, with the database now made, I select "Create" and then pick "Query Design" from the Menu:



A blank query should be created. I add the database to sort through with SQL:



Next, we want to select the SQL View option, to directly put in the SQL Code:



The code to use for this part of answering question 3 is: "SELECT DISTINCT CountryResidence AS Country FROM EcologicalSurveyData;". With this piece of code, we can see that there are 17 different countries that are represented, of where people live in:



As for the countries that the applicants are from, we use the code: "SELECT DISTINCT CountryFrom AS Country FROM EcologicalSurveyData;" and we can see that there are 26 different countries:



To answer question 4, we have to look at the CountryResidence and CountryFrom columns. We need to pick the specific pieces of data that have the data "United States of America" or "United Arab Emirates" as data, and taking the corresponding Ecological Footprint data and finding the average: For the USA, the code is "SELECT AVG(EcologicalFootprint) AS 'Number_of_earths' FROM `EcologicalSurveyData` WHERE CountryResidence = 'United States of America';".



We can see the average is about 7.67


For the data in the UAE, we just change the last part of the code: "SELECT AVG(EcologicalFootprint) AS 'Number_of_earths' FROM `EcologicalSurveyData` WHERE CountryResidence = 'United Arab Emirates';".



Based on the data given, the United Arab Emirates has the higher Average Ecological Footprint.
